# 图片说明文件

本目录用于存放福建省血管衰老重点实验室博客网站所需的图片文件。

## 需要的图片文件列表

请将以下图片文件放置在此目录（`images/`）中：

- **equipment-2.jpg** - 分子生物学设备照片（建议尺寸：600x400px）

- **equipment-4.jpg** - 动物实验平台照片（建议尺寸：600x400px）

### 2. 人员照片
- **director.jpg** - 洪华山教授个人照片（建议尺寸：400x500px，竖版）

### 3. 团队照片
- **team-photo.jpg** - 团队合影（建议尺寸：1200x600px）

## 图片要求

1. **格式**：建议使用 JPG 或 PNG 格式
2. **质量**：图片应清晰，分辨率适中，文件大小控制在 500KB 以内
3. **内容**：图片内容应与描述相符，体现实验室的专业性和学术氛围
4. **版权**：确保使用的图片具有合法使用权

## 注意事项

- 如果某个图片文件不存在，网页会自动显示占位符提示
- 图片文件名必须与上述列表完全一致（区分大小写）
- 建议对图片进行适当压缩以优化网页加载速度

## 图片优化建议

可以使用以下工具优化图片：
- [TinyPNG](https://tinypng.com/) - 在线图片压缩工具
- [Squoosh](https://squoosh.app/) - Google 的图片压缩工具
